# Release Notes

Este documento é o índice principal das notas de lançamento do ContaBase.
Para o histórico técnico detalhado, consulte o [CHANGELOG.md](CHANGELOG.md).

## Preparada para publicação

- **[v0.1.0-beta.1](docs/releases/v0.1.0-beta.1.md)** — *Beta pública controlada*

  Primeira Beta pública controlada do ContaBase, preparada para publicação. Inclui imagem Docker multi-arch via `ghcr.io/contabase-app/contabase`, artifacts de GitHub Release para Linux `amd64`/`arm64` com checksum obrigatório e build local/source avançado.

  A release ainda não foi publicada. Recorrências/parcelamentos (B4) e pagamento parcial de faturas (B5) permanecem como limitações conhecidas. Backup e teste de restore são obrigatórios antes de usar dados importantes.
