# Contribuindo para o ContaBase

Agradecemos o seu interesse no ContaBase.

## Pull Requests

**Atualmente não aceitamos Pull Requests públicos neste repositório.**

O repositório público do ContaBase recebe atualizações estáticas (snapshots limpos) oriundos do nosso repositório de desenvolvimento privado. O nosso fluxo de trabalho de desenvolvimento oficial não utiliza Pull Requests do GitHub como mecanismo de contribuição aberta no momento.

Se um Pull Request for aberto, ele poderá ser fechado sem revisão.

## Issues (Problemas)

Se a seção de Issues estiver ativada, você pode abrir chamados para:

- Relatar bugs reproduzíveis (inclua passos para reproduzir, comportamento esperado e comportamento real).
- Solicitações de funcionalidades (feature requests) com motivação clara e caso de uso.

Por favor, **nunca** inclua:

- Dados sensíveis (senhas, chaves de API, banco de dados, informações pessoais).
- Configurações específicas do seu ambiente (seu arquivo `.env`, IPs de servidores, URLs internas).
- Capturas de tela que contenham dados financeiros reais ou privados.

## Vulnerabilidades de Segurança

Se você descobrir uma vulnerabilidade de segurança, por favor, **não** abra uma Issue pública. Siga as instruções descritas no [SECURITY.md](SECURITY.md) ou contate os mantenedores diretamente.
